# Carrier

## Description

The UI it’s the "Carriers" page. It presents you with a list of all your current carriers. From there, you can directly change their status, indicate why shipping is free or not, and change their position when presented to customers.

## Behaviors description

#### <mark style="color:purple;">Filter</mark>&#x20;

### ID  input

This is input [(Tables with filters UI kit ) ](https://build.prestashop-project.org/prestashop-ui-kit/?path=/story/tables--with-filters)to enter Carrier ID. It's for filter the list by carrier's ID

### Name input

This is input [(Tables with filters UI kit ) ](https://build.prestashop-project.org/prestashop-ui-kit/?path=/story/tables--with-filters)to enter Carrier's Name. It's for filter the list by carrier's name&#x20;

### Delay  input

This is input [(Tables with filters UI kit ) ](https://build.prestashop-project.org/prestashop-ui-kit/?path=/story/tables--with-filters)to enter Carrier delay. It's for filter the list by carrier's Delay

### Status  dropdown

This is Dropdown ([Dropdown Basic UI kit](https://build.prestashop-project.org/prestashop-ui-kit/?path=/story/dropdowns--basics)) to select _`Yes`_ or _`No` ._&#x20;

* `Yes` shows enabled Carrier&#x20;
* `No` shows disabled carrier&#x20;

### Free Shipping dropdown

This is Dropdown ([Dropdown Basic UI kit](https://build.prestashop-project.org/prestashop-ui-kit/?path=/story/dropdowns--basics)) to select _`Yes`_ or _`No` ._&#x20;

* `Yes` shows Carriers have activated _Free shipping_.&#x20;
* `No` shows Carriers did not activate _Free shipping_

### Postion input

This is input [(Tables with filters UI kit ) ](https://build.prestashop-project.org/prestashop-ui-kit/?path=/story/tables--with-filters)to enter Carrier postion. It's for filter the list by carrier's postion

You can also click on this column moving it up or down. After changing the position, the message “Successful update.” is displayed.

#### <mark style="color:purple;">Dropdown button</mark>

### Edit / Delete button

#### Edit selected CTA ([Dropdown Basic UI kit](https://build.prestashop-project.org/prestashop-ui-kit/?path=/story/dropdowns--basics))

It allows you to have direct access to `General setting`  page&#x20;

#### Delete selected CTA ([Dropdown Basic UI kit](https://build.prestashop-project.org/prestashop-ui-kit/?path=/story/dropdowns--basics))

This action will delete the selected carrier.&#x20;

After clicking on “Delete selected”, a modal with the title “Delete selection” is opened asking to confirm or cancel the action: “Delete the selected item ?”. The action can be canceled by clicking on the cross or on the No button. After clicking on “Yes” button, the selected attributes and all their values are deleted, a successful message is displayed in a alert basic ([Alert basic UI KIT](https://build.prestashop-project.org/prestashop-ui-kit/?path=/story/alerts--basics))  “Successfully deleted.”

### Bulk actions behavior&#x20;

When there are multiple carrie, Bulk Actions dropdown ([Dropdowns basics UI kit](https://build.prestashop-project.org/prestashop-ui-kit/?path=/story/dropdowns--basics)) help manage them and configure the main actions.  \
**Checkboxes** ([Forms checkboxes UI kit](https://build.prestashop-project.org/prestashop-ui-kit/?path=/story/forms--checkboxes)) in each carrier line can be used, if there is need to manage carrier separately and use for Bulk Actions for adjusting the carrier. \
Once clicked on Bulk actions dropdown opens these options:&#x20;

* **Select all** - option to select all countries.
* **Unselect all** - option to unselect all selected countries.
* **Enable selected** - option to enable selected zones.
* **Disable selection** - option to disable selected zones. &#x20;
* **Delete selected** - option to delete selected zones.

#### <mark style="color:purple;">Action button</mark>&#x20;

### Add new country button behavior

This is CTA button with plus icon ([Buttons with icons UI kit](https://build.prestashop.com/prestashop-ui-kit/?path=/story/buttons--buttons-with-icons)) once clicked it opens Add a new general setting page.&#x20;
